# Portfólio — Leonardo Gomes

Portfólio pessoal desenvolvido com React + Vite. Responsivo, com suporte a tema claro/escuro.

## Como rodar localmente

**Pré-requisito:** Node.js 18+

```bash
git clone https://github.com/leeoze/portfolio-LG.git
cd portfolio-LG
npm install
npm run dev
```

Acesse: `http://localhost:5173`


---

## Stack

- **React 18** — componentes funcionais e Hooks
- **Vite** — bundler
- **CSS Modules** — estilização com escopo local
- **Context API** — gerenciamento do tema
